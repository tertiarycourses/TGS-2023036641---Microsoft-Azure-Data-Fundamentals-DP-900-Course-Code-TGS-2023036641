# Tools Guide

## Required Tools

| Tool | Purpose |
| --- | --- |
| Microsoft Learn | Study guide, learning modules, and sandbox exercises. |
| Azure portal | Review and create Azure data services. |
| Azure Cloud Shell | Run Azure CLI and SQL-related commands where appropriate. |
| Azure Data Studio | Query relational databases and inspect schemas. |
| Power BI Desktop | Build basic data models and visualizations. |

## Optional Tools

| Tool | Purpose |
| --- | --- |
| Azure Storage Explorer | Inspect Blob, Files, Queues, and Tables. |
| SQL Server sample databases | Practice relational concepts. |
| diagrams.net | Draw data architecture diagrams. |
| Spreadsheet tool | Compare services and workload patterns. |

At the end of class, check for:

- Azure SQL databases.
- Storage accounts.
- Cosmos DB accounts.
- Databricks workspaces.
- Fabric trial workspaces or lakehouses.
- Resource groups created only for training.
