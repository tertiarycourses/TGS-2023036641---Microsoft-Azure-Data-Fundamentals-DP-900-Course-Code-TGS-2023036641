# TGS-2023036641 - Microsoft Azure Data Fundamentals (DP-900) Course Code: TGS-2023036641

> Course: Microsoft Azure Data Fundamentals (DP-900)  
> Course Code: TGS-2023036641  
> Register here: https://www.tertiarycourses.com.sg/wsq-microsoft-azure-data-fundamentals-dp-900.html

Hands-on Azure Data Fundamentals labs for learners preparing to understand core data concepts and Microsoft Azure data services. The labs cover structured, semi-structured, and unstructured data; transactional and analytical workloads; relational databases; Azure SQL; open-source databases on Azure; Azure Storage; Azure Cosmos DB; data ingestion and processing; Microsoft Fabric; Azure Databricks; real-time analytics; and Power BI.

## How to Use These Labs

1. Read the learner guide before starting the first lab.
2. Complete the labs in order because later labs reuse concepts from earlier labs.
3. Use Microsoft Learn, an Azure free account, or an instructor-provided Azure subscription.
4. Prefer free tiers and sandbox environments where possible.
5. Clean up any resources created during class.
